# 🎹 Harmonium

> *Your keyboard is an instrument. It always was.*

A vintage harmonium that lives entirely in your browser, no installs, no accounts, no nonsense. Just open the page, press a key, and make music.

---

## ✨ What is this?

This is a fully playable **harmonium** built in pure HTML, CSS, and JavaScript. That means it runs on **literally anything with a browser**, your laptop, your phone, a tablet, an ancient family PC in the corner of a room. No app store. No setup. No build step. Just one file.

It was also a small proof of something: that a mundane QWERTY keyboard , the thing you use to write emails and argue on the internet can become a musical instrument with the right 300 lines of code.

---

## 🎵 Features

| Feature | Description |
|---|---|
| 🎹 **3 Octave Range** | Full C3–C6 range across white and black keys |
| ⬆️ **Dynamic Octave Shifting** | Shift the mapped range up or down with `← →` arrow keys |
| 🔇 **Sustain Mode** | Hold notes like a real harmonium — toggle with `Space` |
| 🕉️ **Drone Mode** | Lock any note as a continuous drone in the background |
| 🔴 **Record & Playback** | Record your sessions and replay them with one click |
| 🔊 **Volume Control** | Smooth volume slider |
| 📱 **Touch Support** | Fully playable on mobile and tablet screens |

---

## 🖥️ How to Play

```
White keys  →  Z  X  C  V  B  N  M  ,  .  /
Black keys  →  S  D     G  H  J     L  ;
Sustain     →  Space
Octave up   →  →  (right arrow)
Octave down →  ←  (left arrow)
```

Or just tap/click the keys on any touch device.

## 🛠️ Tech Stack

```
HTML5          — structure
CSS3           — vintage wooden aesthetics
JavaScript     — keyboard mapping, audio logic
Tone.js        — Web Audio synthesis engine
Google Fonts   — Playfair Display, IM Fell English
```

No frameworks. No bundlers. No `node_modules` folder the size of a small country. Just a single `.html` file you could read end to end in an afternoon.

---

## 🎨 Design Philosophy

The UI is built to feel like a **real instrument** — warm wood tones, ivory keys, brass hardware. Because if you're going to play music, the interface should feel like it belongs in a music room, not a SaaS dashboard.

The sound engine uses **sawtooth wave oscillators** run through a polyphonic synth — the same waveform that gives reed instruments their characteristic buzzy warmth. It's not perfect. It's not trying to fool anyone. But it's surprisingly fun to play.

---

## 💡 The Bigger Idea

Every keyboard is already an instrument. The keys have weight, rhythm, muscle memory. Programmers type on them for hours a day and never think of them as expressive. This project is a small reminder that **any interface can become musical** if you map it differently.

A spreadsheet could be a drum machine. A search bar could be a theremin. A login form could be a chord progression.

The web is weird and that's what makes it wonderful.
